## ğŸš€ Installation & Execution

To run the application on your local machine, follow these steps:

### Prerequisites
* **Java Development Kit (JDK):** Version 17 or higher.
* **Maven:** For dependency management.
* **IDE:** IntelliJ IDEA (Recommended) or Eclipse.

### How to Run
1.  Clone or download this repository.
2.  Open the **`KidTask v1.0`** folder as a project in your IDE.
3.  Let Maven download the necessary dependencies (JavaFX, Gson, etc.).
4.  Navigate to the entry point of the application:
    `src/main/java/project/Launcher.java`
5.  Right-click on **`Launcher.java`** and select **Run**.
